# Website Project Presentation

## Project Title
**Le Coin Vert – Gardening Equipment Online Store**

## Description
Le Coin Vert is a web-based project designed to sell gardening tools and equipment online.  
The website provides a simple, user-friendly interface that allows users to browse products, add them to a cart, and contact the store.

## Objectives
- Create a responsive and attractive website
- Display gardening products with prices and images
- Implement a dynamic shopping cart
- Allow users to contact the store easily

## Technologies Used
- **HTML5** – structure of the website  
- **CSS3 / Bootstrap** – styling and responsive design  
- **JavaScript** – cart management and interactivity  
- **LocalStorage** – save cart data in the browser

## Website Structure
- **Home Page** – introduction and featured products  
- **Products Page** – list of available gardening tools  
- **Cart Page** – view and manage selected products  
- **Contact Page** – contact form for customers  

## Main Features
- Product listing with name, price, and image
- Add / remove products from cart
- Automatic total price calculation
- Responsive design for mobile and desktop
- Contact form validation
